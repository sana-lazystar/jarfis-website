AI와 함께 소프트웨어를 만드세요 — 슬래시 명령어 하나, 11명의 전문 에이전트, Human Gate가 있는 구조화된 Phase.

## 사전 요구사항

시작하기 전에 다음을 준비하세요:

- **Claude Code** (Anthropic) — JARFIS는 Claude Code 안에서 슬래시 명령어 워크플로우로 동작합니다
- API 접근 권한이 있는 Claude 계정
- **Git** >= 2.x (Phase 0의 브랜치 관리를 위해)
- [JARFIS GitHub 저장소](https://github.com/sana-lazystar/jarfis) 클론 또는 접근 가능 상태

## 설치

JARFIS는 GitHub 저장소의 `install.sh` 스크립트로 설치합니다. 터미널에서 다음을 실행하세요:

```bash
bash install.sh
```

<Callout>
  **참고:** `/jarfis:install` 명령어는 없습니다. 설치는 반드시 `bash install.sh`로만 진행합니다.
</Callout>

## 첫 워크플로우 실행

프로젝트 디렉토리에서 Claude Code를 열고 워크플로우를 시작하세요:

```
/jarfis:work Build a user authentication system with JWT tokens
```

JARFIS가 요청을 분류(Phase T — Triage)한 뒤, 각 단계를 전문 AI 에이전트가 담당하는 구조화된 Phase를 안내합니다.
