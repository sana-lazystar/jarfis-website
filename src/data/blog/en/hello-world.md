# JARFIS v1.0 is Here

We're thrilled to announce **JARFIS v1.0** — the open-source AI agent framework that orchestrates specialized agents to build production-ready software, autonomously.

## What is JARFIS?

JARFIS (Just Another Really Fantastic Intelligent System) is a 100% open-source framework that brings a team of specialized AI agents together:

- **PO Agent** — Defines requirements and breaks down tasks
- **UX Agent** — Designs wireframes and user flows
- **Frontend Agent** — Builds components and pages
- **QA Agent** — Tests and verifies quality

These agents collaborate just like a real development team — but autonomously, without vendor lock-in, and on your own infrastructure.

## Why We Built This

Traditional AI coding assistants are powerful, but they're single agents making suggestions. JARFIS is different: it's a **team** that executes.

We built JARFIS because we believe the future of software development is:
1. **Autonomous** — Agents that don't just suggest, but do
2. **Transparent** — Open source, auditable, no black boxes
3. **Yours** — Self-hosted, your LLM, your data

## Get Started

```bash
npm install -g @jarfis/cli
jarfis init my-project
cd my-project
jarfis run
```

That's it. JARFIS takes over, orchestrates the agents, and delivers production-ready output.
